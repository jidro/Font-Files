### 字体功能调用及实现

font-feature 属于 Opentype 中的高级排版功能，也就是可以让字体排版更加精致，符合实际设计、开发需求，可根据特定设计需求选择不同的字体形态。

大部分软件都可以通过 Opentype 面板访问此功能。
