# 极影毁片圆
## 中文
> **说明：**
> 具体权利请搜索及联系原作者。
极影毁片圆是以小衫圆体、思源黑体所修改而成的字体。
本字体以jf open粉圆为底本，
使用江城圆体3.0、狮尾圆体SC作为增补，
补全了Unicode的中日韩统一表意文字
